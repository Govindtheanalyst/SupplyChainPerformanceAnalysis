# 📦 Supply Chain Performance Analysis

## 📖 Project Overview

This project analyzes a global supply chain dataset containing over **180,000 orders** to uncover insights into sales performance, product profitability, customer demand, and logistics efficiency.

The project demonstrates an end-to-end data analytics workflow, including:

* Data Cleaning with Python (Pandas)
* Exploratory Data Analysis (EDA)
* SQL Business Analysis using PostgreSQL
* Interactive Dashboard Development using Power BI

---

# 🎯 Business Problem

Supply chain operations generate large amounts of transactional data, but without proper analysis it is difficult to identify:

* Which products generate the highest revenue?
* Which markets contribute the most sales?
* Where are shipping delays occurring?
* Which shipping methods perform best?
* Which product categories are most profitable?

The objective of this project is to transform raw operational data into actionable business insights through data analysis and interactive dashboards.

# 🧹 Data Cleaning (Python)

The dataset was cleaned and prepared using Pandas.

Cleaning tasks included:

* Removed duplicate records
* Standardized column names to snake_case
* Checked for missing values
* Corrected data types
* Created derived date columns
* Exported cleaned dataset for SQL and Power BI

# 📈 Power BI Dashboard

The dashboard consists of **three interactive pages**.

## 1️⃣ Executive Overview

Provides a high-level summary of business performance.

Features:

* KPI Cards
* Monthly Sales & Profit Trend
* Revenue by Market
* Orders by Shipping Mode
* Delivery Status Distribution

---

## 2️⃣ Logistics & Delivery Analysis

Focuses on supply chain efficiency.

Features:

* Shipping Delay KPIs
* Average Delay by Region
* Delay by Product Category
* Shipping Mode Performance
* Global Late Delivery Hotspots

---

## 3️⃣ Product & Revenue Analysis

Analyzes product performance and profitability.

Features:

* Revenue by Category
* Profit by Category
* Top Products by Revenue
* Revenue vs Profit Comparison
* Product Performance KPIs

---

# 📌 Key Insights

* Europe generated the highest overall revenue.
* Standard Class handled the majority of shipments.
* Late deliveries represented a significant portion of total orders.
* Certain product categories generated high revenue but relatively lower profit.
* Shipping delays varied considerably across regions and shipping methods.
